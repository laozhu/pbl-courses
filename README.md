# 🚀 PBL 学习乐园

欢迎来到我们的学习乐园！这里是一个用 **PBL** 方式来学习的地方。

## 🌱 什么是 PBL？

PBL 的全名是 **Project-Based Learning**，中文叫「**项目式学习**」。

简单来说，就是——

> **我们不只是听老师讲、背知识，而是通过「做一个真实的小项目」来学会新本领。**

打个比方 🍳：

- 想学做菜，最好的办法不是背菜谱，而是**真的下厨做一盘菜**。
- 想学骑自行车，光看别人骑没用，得**自己骑上去试**。

PBL 就是这样：**边做边学，做完一个小任务，你就真的学会了一样本领。** 🎉

## 📦 这个项目是怎么组织的？

我们把学习内容拆成一个一个的「**任务（task）**」。

所有任务都放在 `tasks/` 文件夹里，每个任务是一个小文件夹，名字前面有**编号**，方便按顺序完成：

```
pbl-learning/
├── README.md              ← 你正在看的这个文件
└── tasks/                 ← 所有任务都放在这里
    └── 01-learn-git/      ← 第 1 个任务：学习 Git 和 GitHub
        ├── README.md      ← 任务说明（怎么闯关）
        └── my-work/       ← 你完成的作品放在这里 📁
```

命名规则很简单：

- `01-`、`02-`、`03-`…… 是**任务的顺序**，从小到大一个个来。
- 后面的名字（比如 `learn-git`）告诉你**这个任务在学什么**。

## 🧭 怎么开始？

1. 打开 `tasks/` 文件夹。
2. 找到**编号最小**、还没完成的任务（现在是 `01-learn-git`）。
3. 打开它里面的 `README.md`，跟着一步步做。
4. 做完后，把你的作品放进那个任务的 `my-work/` 文件夹里。
5. 回到这里，把上面清单里的状态改成 ✅ 完成！
