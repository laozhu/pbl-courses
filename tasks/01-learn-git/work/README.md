# 📁 学员作品（work）

这个文件夹是用来**存放所有学员完成的作品**的。

每位学员都要在这里建一个自己的文件夹。

提交规则：

- 文件夹名必须包含你的 GitHub 用户名。
- 推荐直接使用你的 GitHub 用户名，比如 `xiaoming`。
- 不要把作品直接放在 `work/` 根目录里。

```
work/
├── _template/
│   └── 完成记录.md
├── xiaoming/
│   ├── about-me.md
│   ├── my-pr.png
│   └── 完成记录.md
└── xiaohong/
    ├── about-me.md
    ├── my-pr.png
    └── 完成记录.md
```

做任务 01 的时候，请把这些东西放进来：

- ✅ `你的GitHub用户名/about-me.md` —— 你的自我介绍卡片
- 📸 `你的GitHub用户名/my-pr.png` —— 你的 PR 页面截图（可选）
- 📝 `你的GitHub用户名/完成记录.md` —— 从 `_template/完成记录.md` 复制后填写

> 全部放好后，这就是你学会 Git 的「小小纪念品」啦！🥳
